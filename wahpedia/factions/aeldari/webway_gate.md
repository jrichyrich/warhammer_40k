#### **Abilities**
*   **Ambush From the Webway:** While a friendly AELDARI unit is being set up on the battlefield from Strategic Reserves, if it is set up wholly within 9" of this model, it can be set up more than 9" away from enemy models (standard rules apply), but it can also be set up within 9" of enemy models provided it is not within Engagement Range.
*   **Fortification:** This model cannot move, cannot be placed in Reserves, and cannot be destroyed by normal attacks (it is treated as terrain once deployed).
*   **Wraithbone Screen:** Models have a 4+ invulnerable save while within 6" of this model.

**Points:** 220 pts
