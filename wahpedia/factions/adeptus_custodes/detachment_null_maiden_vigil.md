# Detachment: Null Maiden Vigil

This detachment focuses on the anti-psychic and morale-disrupting capabilities of the Sisters of Silence.

## Detachment Rule: Creeping Dread (Aura)
In the Battle-shock step of your opponent’s Command phase, enemy **PSYKER** units or those below Starting Strength within 12" of an **ANATHEMA PSYKANA** model must take a Battle-shock test (at -1 if Below Half-strength).
*   **Special:** **PROSECUTORS** units have the **BATTLELINE** keyword in this detachment.

## Enhancements
*   **Enhanced Voidsheen Cloak (10 pts):** **ANATHEMA PSYKANA** model only. Subtract 1 from Damage (or set to 1 if from a **PSYKER**).
*   **Huntress’ Eye (15 pts):** **ANATHEMA PSYKANA** model only. Force a Battle-shock test on an enemy within 12".
*   **Oblivion Knight (25 pts):** **ANATHEMA PSYKANA** model only. Unit adds 1 to Hit rolls (and 1 to Wound rolls against **PSYKERS**).
*   **Raptor Blade (5 pts):** **ANATHEMA PSYKANA** model only. +1 Attacks, Strength, and Damage (+2 if near a Battle-shocked **PSYKER**).

## Stratagems
*   **DESPERATION’S PRICE (1CP):** (Psychic use) Enemy **PSYKER** must take a Leadership test or suffer 3 mortal wounds and Battle-shock.
*   **WITCH HUNTERS (1CP):** (Shooting/Fight phase) Unit gains **[LETHAL HITS]** or **[SUSTAINED HITS 1]** but can only target **PSYKERS**.
*   **ANATHEMA BLADEMASTERY (1CP):** (Fight phase) **VIGILATORS** unit re-rolls Hit rolls (and Wounds vs **PSYKERS**/Battle-shocked).
*   **PSY-CHAFF VOLLEY (1CP):** (Shooting phase) **PROSECUTORS** unit marks a target; others get +1 AP against it and target is -1 to Hit if **PSYKER**.
*   **PURGATION SWEEP (1CP):** (Shooting phase) **WITCHSEEKERS** unit adds +1 to Torrent attacks (+2 if vs **PSYKER**).
*   **PSYCHIC ABOMINATIONS (1CP):** (Enemy Shooting phase) Unit gains **Stealth** and can only be targeted within 12" by **PSYKERS**.
